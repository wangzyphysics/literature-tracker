今日摘要

总览：今日凝聚态与计算材料条目呈现三条主线：机器学习与物理约束计算覆盖室温储氢高熵合金、铁钼金属间化合物、铀锆液态合金、超硬硼碳氮金属以及交换关联泛函；磁性与拓扑电子学集中在交替磁体、 kagome 体系 GdV₆Sn₆、单层过渡金属硫族化物、Ni₂MnGa、Sr₁₋ₓNdₓCuO₂ 和金刚石结构 EuTi₂Al₂₀。超导方向同时出现常压 La₃Ni₂O₇ 双层镍酸盐薄膜、非中心对称弱关联金属的含自旋三重态节点超导、三维超导交替磁体交叉表面平带，以及基于第一性原理的超硬硼碳氮金属临界温度计算。

热点：热点一：交替磁体与自旋输运 → 通过轨道通量激活 kagome 交替磁体的贝里曲率、利用 Lifshitz 转变产生开轨道自旋过滤和持久自旋流、讨论三维超导交替磁体的交叉表面平带 → 典型工作为[12]、[15]、[20]，并由[10]给出交替磁性与超导的历史脉络。热点二：镍酸盐与非常规超导 → 在常压 La₃Ni₂O₇ 薄膜中寻找双层镍酸盐超导增强，同时把 Sr₁₋ₓNdₓCuO₂ 无限层铜酸盐的反铁磁量子临界性作为对照体系 → 典型工作为[31]、[33]、[14]。热点三：物理约束机器学习与材料建模 → 人工神经网络联用密度泛函理论和实验优化室温储氢高熵合金，导数信息学习交换关联泛函，核化分子动力学与通用机器学习势用于主动学习原子间势、熔点和铀锆液体黏度 → 典型工作为[1]、[24]、[48]、[49]、[51]、[59]。热点四：铁电、磁电和神经形态器件 → 扫描探针尖端下铁电翻转从场限制跨入电流限制，二维端铁电忆阻器和畴壁神经元连接智能计算，柔曲塞曼耦合在共线磁电反铁磁体中诱导室温局域多铁序 → 典型工作为[16]、[18]、[19]、[13]。
