热点：热点一：拓扑与非共线磁性器件 → 通过钴掺杂锰三锡稳定非共面反铁磁自旋纹理并放大实空间贝里曲率，同时用三维曲面空心磁性纳米壳和反向磁场诱导笼状微结构调控曲率各向异性与电磁吸收 → 典型工作为钴掺杂锰三锡宽温区拓扑霍尔效应、直写三维曲面磁性纳米壳、磁场反转诱导笼状吸波体。热点二：铁电/离子/自旋神经形态硬件 → 以铁电异质结实现自供能宽谱探测、谱调谐和光电突触，以木聚糖纳流体膜忆阻器实现光控离子神经网络，以合成反铁磁器件在辐照下重构海表温度 → 典型工作为可重构铁电异质结、光调制纳流体忆阻器、辐照耐受合成反铁磁神经形态器件。热点三：高温与极端环境电子材料 → 通过超支化介电聚合物网络抑制高温漏电并提高储能密度，通过缺陷诱导静电锚定增强功率器件封装的极端温度可靠性，通过空心玻璃微球/有机硅夹层复合物兼顾电磁屏蔽与超低电导 → 典型工作为二百五十摄氏度巨储能聚合物、缺陷锚定封装、低导电屏蔽复合物。热点四：界面催化与孔结构精确化 → 用双极轴互生铁电内建电场分离光生载流子，用层状共价有机结构驱动质子耦合电子转移析氧，用电化学退火铂铜互锁网络优化氧还原吸附能，用介孔限域单原子网络稳定类芬顿活性位 → 典型工作为铁电全分解水、无金属析氧催化、铂铜氧还原、单原子介孔类芬顿催化。
