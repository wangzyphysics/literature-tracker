今日摘要

总览：今日文献中，深度学习分子动力学揭示了双层六方氮化硼中超快滑移铁电翻转的原子机制；等变神经网络密度矩阵学习（如MACE风格架构）加速自洽场计算与分子振动光谱预测成为计算化学方法学焦点；交错磁体（altermagnet）中量子几何驱动的巨大自旋磁化与反常热霍尔效应等输运现象持续受到关注，涵盖Co掺杂FeSb₂与d波交错磁体等具体材料体系。

热点：热点一：等变神经网络嵌入电子结构计算 → 通过等变密度矩阵学习加速SCF迭代，或以等变性质面预测红外/拉曼光谱 → 体现为文献[11]的加速SCF工作流与文献[12]的甲醇分子振动光谱。热点二：滑移铁电/二维铁电的微观机制与调控 → 利用深度势能面探索双层h-BN中层间滑移导致的极化翻转路径与动力学 → 文献[1]结合深度学习分子动力学给出超快开关的直接模拟证据。热点三：交错磁体（altermagnet）中的量子几何效应 → 将贝里曲率、量子度规与自旋磁化、反常霍尔/热霍尔输运关联 → 文献[45]预言巨大自旋磁化，文献[47]提出d波交错磁体中本征反常热霍尔效应作为量子度规的指纹，文献[41]则从光学响应识别Co掺杂FeSb₂的金属型交错磁性。
