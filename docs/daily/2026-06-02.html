今日摘要

总览：今日凝聚态与计算材料条目集中在机器学习势能面、长程静电、动力学蒙特卡洛与分子动力学：钨中氢输运、二氧化钛极化子迁移、碳化硅多型熔化、热力学相图自由能建模和双层堆垛材料性质预测构成一条从原子势到工艺参数的计算链。量子材料侧的主线是交替磁性、莫尔与滑移铁电、反铁磁拓扑态和非常规输运，覆盖氟化锰、铁硒双层、铈镍砷氧、石墨烯/六方氮化硼莫尔超晶格、三菱相二硒化钨双层、砷化铕锡和重费米子铁磁体等体系。另有拓扑超导碲硒化铁中马约拉纳零模谱分解、铜氧化物高温超导光稳态利夫希茨转变、钇铁石榴石纳米结构自旋波参量放大、磁绝缘体分数化激发的自旋塞贝克读出等谱学与输运主题。

热点：热点一：材料机器学习从性质拟合转向物理约束生成与动力学外推 → 用势能面光滑性指标选择机器学习原子间势架构、可微分粒子网格埃瓦尔德结合笛卡尔张量消息传递学习长程静电与偶极响应、主动学习耦合半导体器件优化 → 典型工作包括势能面光滑性指导势函数、钨中氢输运的深度学习动态动力学蒙特卡洛、二氧化钛极化子输运的机器学习分子动力学、热力学相图自由能建模加速。热点二：交替磁性成为反铁磁自旋电子学的核心线索 → 关注零净磁矩材料中的动量依赖自旋劈裂、外尔节线环、轨道/自旋霍尔量子化、堆垛可切换磁序以及局域四电子对交替磁性的抑制 → 典型体系为氟化锰、铁硒双层、三角晶格交替磁体、铈镍砷氧，并伴随电子霍尔黏滞作为反铁磁隐藏诊断量。热点三：二维层状材料的堆垛、扭转和滑移自由度继续把结构动力学转化为电子与铁电功能 → 单取向石墨烯/六方氮化硼出现铁电滞回，滑移扭转双层产生莫尔应变斯格明子，化学气相沉积三菱相二硒化钨双层的无序动力学和多畴动力学控制滑移铁电，扭转平面转子磁体出现非共线自旋纹理与九十度畴壁。热点四：拓扑与非平衡量子现象向可读出、可调控平台收敛 → 机器学习辅助谱分解马约拉纳零模，反铁磁拓扑绝缘体砷化铕锡成像磁驱动电子态重构，高温超导铋锶钙铜氧中光稳态利夫希茨转变，螺旋边缘态安德烈夫自旋量子比特和数字量子处理器上的共形弗洛凯动力学共同指向谱学、输运和量子信息接口。
