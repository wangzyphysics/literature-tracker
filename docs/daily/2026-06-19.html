今天的推进集中在用机器学习分子动力学追踪铁电极化动力学，但[1]目录未披露是否覆盖 BaTiO3、PbTiO3、NbOI2，也未说明采用 equivariant GNN、MACE 还是 NEP。通用力场与合金势评估从DFT误差转向实验可观测量和跨成分迁移，直接牵动 MACE、CHGNet、M3GNet、NEP 在 BaTiO3、CrI3、CrSBr 等软模或磁弹耦合体系上的可信度。d波交替磁工作把 Boltzmann 约束用于区分自旋劈裂与动量弛豫，和 DFT+U 处理 CrI3、CrSBr、MnTe、RuO2 类磁性体系输运参数反演相接。DFT超敏感性研究指出训练标签本身的不稳定会污染 ML 势和性质模型，尤其影响 NbOI2、BaTiO3 的极化势垒以及 CrI3、CrSBr 的磁交换参数学习。

今日摘要

总览：今日条目集中在机器学习原子势与物理约束神经网络：机器学习分子动力学用于铁电体极化翻转，通用力场被拿去同实验结构、溶剂化和水交换数据对照，密度泛函理论的数值超敏感性则被作为训练数据可靠性的瓶颈处理。自旋与拓扑方向由交替磁性贯穿，体系覆盖碲化铋族范德瓦耳斯异质双层、二维铯铬锑单层、锰锡拓扑反铁磁体、溴硫化铬双层、笼目超导体和石墨烯自旋极化，现象包括无净磁化马约拉纳模、量子自旋霍尔相、横向约瑟夫森二极管效应、轨道霍尔效应和通道化超电流。

热点：热点一：机器学习势能面从“拟合能量”转向“实验可检验动力学” → 用机器学习分子动力学、跨成分合金势、密度泛函理论超敏感性修正来检验铁电极化动力学、含镁水溶液溶剂化和水交换、合金全成分区间结构 → 典型工作为铁电体极化动力学、通用机器学习力场实验评测、合金跨成分势、含镁水溶液神经网络势。热点二：交替磁性成为自旋电子学与拓扑超导的主线 → 通过玻尔兹曼约束反演自旋劈裂和动量弛豫、表面铁磁层诱导大自旋劈裂、无自旋轨道耦合的螺旋交替磁体产生马约拉纳模、交替磁体中实现场零横向约瑟夫森二极管效应 → 典型工作为四叶型交替磁体输运反演、体反铁磁表面大自旋劈裂、二维铯铬锑交替磁基态、交替磁体量子自旋霍尔效应。热点三：物理约束学习正在向算子、控制和材料本构扩展 → 用注意力物理约束神经网络、两阶段进化超参数优化、宽度学习系统、柯尔莫哥洛夫—阿诺德网络和凸神经表示求解耦合流动输运、瞬态福克—普朗克方程、塑性屈服函数和电池内部状态 → 典型工作为任意参数初值福克—普朗克联合求解、凸神经屈服函数发现、长跨桥梁距离感知深度算子学习、电池状态自回归神经代理。
