今日摘要

总览：今日题目集中在量子材料的磁性拓扑与机器学习建模两条主线：二硒化铟分子束外延多晶型选择、二硫化钼可逆手性翻转产生肖流、钒四氰乙烯磁振子扩散、二层六方相铌硒硫多带超导，以及钕硼钨氧螺旋反铁磁体的伊辛超临界磁热响应。方法侧突出注意力基础模型、等变图神经网络、物理增强柯普曼图卷积、字典学习解析延拓、光子可训练测量和拓扑持久图，用于量子态表示、光谱筛选、电子声子耦合、显微组织反演与偏微分方程学习。

热点：热点一：交错磁性与自旋输运 → 通过共振非弹性Ｘ射线散射识别ｄ波交错磁性、利用表面和薄膜对称性降低投影交错磁性、以超快光学耦合铁电与交错磁开关 → 典型工作包括ｄ波交错磁性、表面和薄膜中的投影交错磁性、铁电－交错磁超快切换。热点二：手性、磁振子和反铁磁斯格明子器件 → 竞争自旋力矩驱动反铁磁斯格明子双向运动，磁振子－位错声子杂化调控磁绝缘体动力学，二硫化钼手性翻转在无磁体条件下产生自旋流，磁异质结构优化太赫兹发射 → 典型工作包括反铁磁斯格明子双向运动、磁振子－位错声子杂化、二硫化钼手性自旋流。热点三：超导与拓扑量子流体 → 腔增强超导、朗道能级耦合常规超导体中的排斥相互作用诱导拓扑超导、超电流操控磁畴壁、韦尔金属体等离激元的量子几何和拓扑 → 典型工作包括腔增强超导、排斥相互作用驱动拓扑超导、多带超导二层六方相铌硒硫。热点四：材料机器学习从经验拟合转向物理约束生成与反演 → 等变图神经网络预测光谱，广义形变势结合机器学习计算半导体电子声子耦合和热电输运，模拟数据训练网络由微结构形貌反演材料参数，字典学习重构条件良好的解析延拓问题 → 典型工作包括量子态注意力基础模型、材料光谱等变图网络、电子声子耦合机器学习。
