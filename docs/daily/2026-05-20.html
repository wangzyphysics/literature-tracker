今日摘要

总览：今日文献呈现机器学习与凝聚态物理深度交叉的格局：图神经网络被用于预测磁体居里温度并强调数据策展的关键作用（工作[1]），深度学习声子辅助光谱（工作[7]）和机器学习势模拟层状氧化物输运（工作[12]）及LiF中锂扩散（工作[23]）推进了有限温度与缺陷物理的计算效率；量子几何与非线性输运（工作[30]）、交错磁体中d波（工作[29]）与g波（工作[44]）自旋织构、以及二维磁体中非局域光伏效应探测隐藏量子度规响应（工作[4]）构成量子材料前沿。

热点：热点一：量子材料中的几何与拓扑响应 → 通过非线性/非局域输运探测量子度规与贝里曲率 → 工作[4]利用非局域光伏效应读取二维磁体隐藏量子度规，工作[30]揭示耗散塑造的量子几何在非线性输运中的角色。热点二：机器学习势与多尺度模拟 → 以MACE等框架实现从头算精度的大尺度动力学 → 工作[12]对NMC811正极材料、工作[23]对LiF中环形集体扩散的机器学习势研究，工作[38]将耦合簇路径积分分子动力学加速至'on-the-fly'。热点三：量子-经典交叉的磁子-光子耦合与量子电池 → 利用深度学习代理模型与腔量子电动力学 → 工作[3]以HPC驱动ML代理模拟磁子-光子动力学，工作[8]用深度学习方法优化驱动腔-海森堡自旋链量子电池充电效率，工作[40]在层状反铁磁中实现相干太赫兹磁子-声子三波混频。
