今日摘要

总览：今日条目集中在自旋—极化—拓扑耦合和机器学习材料计算两条主线：α-MnTe、CuF₂单层、FeF₂、RuO₂、混合氯化锰与铁电交错磁陈绝缘体围绕交错磁自旋劈裂、手性磁振子、反常霍尔电导、可电控陈数展开。计算侧从CrI₃单层应变交换、SmCo-1:7矫顽力微磁数据、AlScN:硼组合薄膜、PbTiO₃居里温度偏差，到电子哈密顿量迁移学习、保自旋对称反向流、推理时强化学习材料生成，显示物理约束神经网络正进入磁性、铁电和强关联电子结构的定量环节。

热点：交错磁性与极化拓扑 → 通过极性畸变、应变、轨道自由度和低维边界态调控自旋劈裂、反常霍尔电导、磁振子非简并与陈数 → 典型工作包括α-MnTe极性畸变调控反常霍尔电导、CuF₂单层手性磁振子与回旋声子、RuO₂应变驱动自旋劈裂、FeF₂磁振子劈裂、边缘自旋伽伐尼效应和铁电交错磁陈绝缘体。物理约束机器学习材料建模 → 将等变图神经网络、不确定性感知神经网络、微磁高通量数据、原子势叠加特征和推理时强化学习用于交换参数、矫顽力、磁性材料性质、电子哈密顿量与开放式材料生成 → 典型工作包括应变CrI₃动态磁交换预测、SmCo-1:7纳米结构矫顽力学习、磁性材料不确定性建模、电子哈密顿量迁移学习和开放材料生成。强关联与量子几何 → 围绕莫尔局域波函数几何、三角哈伯德带填充、重费米子量子临界、量子霍尔与体光伏响应，追踪铁磁性、超导、近藤效应和外禀量子几何贡献 → 典型工作包括莫尔体系几何诱导铁磁性、扭转三层WSe₂四分之一和半填充附近超导、CeSiI压力驱动反铁磁量子临界、CeRh₆Ge₄铁磁量子临界近藤效应和四极体光伏效应。自旋神经形态与磁子器件 → 用射频广播远程写入自旋电子神经网络权重、非线性波神经元集成磁子神经电路、二维铁磁器件自旋轨道矩开关，把磁化动力学从存储单元推向可编程计算单元 → 典型工作包括射频编程自旋神经网络、非线性磁子神经电路和二维铁磁自旋轨道矩翻转。
