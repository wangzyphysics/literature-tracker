# Longitudinal-field fidelity susceptibility analysis of the $J_1$-$J_2$ transverse-field Ising model around $J_2/J_1 \approx 0.5$

## Abstract
arXiv:2601.01893v1 Announce Type: new Abstract: The square-lattice $J_1$-$J_2$ transverse-field (TF) Ising model was investigated with the exact diagonalization (ED) method. In order to analyze the TF-driven phase transition, we applied the longitudinal-field fidelity susceptibility $\chi^{(h)}_F$, which is readily evaluated via the ED scheme. Here, the longitudinal field couples with the absolute value of the magnetic moment $|M|$ rather than the raw $M$ so that the remedied fidelity susceptibility exhibits a peak around the critical point; note that the spontaneous magnetization does not appear for the finite-size systems. As a preliminary survey, the modified fidelity susceptibility $\chi^{(h)}_F$ is applied to the analysis of criticality for $J_2=0$, where a number of preceding results are available. Thereby, properly scaling the distance from the multi-criticality, $\eta=0.5-J_2$, the $\chi^{(h)}_F$ data were cast into the crossover-scaling formula, and the multi-critical exponent for $\chi_F^{(h)}$ is estimated. The result is cross-checked by the numerically evaluated $\beta$-function behavior.

## 摘要（中文）
arXiv:2601.01893v1 公告类型：新 摘要：用精确对角化 (ED) 方法研究了方晶格 $J_1$-$J_2$ 横向场 (TF) Ising 模型。为了分析 TF 驱动的相变，我们应用了纵向场保真度磁化率 $\chi^{(h)}_F$，它可以通过 ED 方案轻松评估。这里，纵向场与磁矩 $|M|$ 的绝对值耦合，而不是与原始 $M$ 耦合，使得修正后的保真度磁化率在临界点附近呈现峰值；请注意，有限尺寸系统不会出现自发磁化。作为初步调查，修改后的保真度敏感性 $\chi^{(h)}_F$ 应用于 $J_2=0$ 的关键性分析，其中有许多先前的结果可用。由此，适当缩放与多临界性的距离$\eta=0.5-J_2$，将$\chi^{(h)}_F$数据代入交叉缩放公式，并估计$\chi_F^{(h)}$的多临界指数。结果通过数值评估的 $\beta$ 函数行为进行交叉检查。
