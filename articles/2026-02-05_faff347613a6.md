## Abstract
arXiv:2602.03857v1 Announce Type: new Abstract: This paper presents a high-performance framework for three-dimensional gravity modeling and inversion implemented in Julia, addressing key challenges in geophysical modeling such as computational complexity, ill-posedness, and the non-uniqueness inherent to gravity inversion. The framework adopts a data-space inversion formulation to reduce the dimensionality of the problem, leading to significantly lower memory requirements and improved computational efficiency while maintaining inversion accuracy. Forward modeling and inversion operators are implemented within a backend-agnostic kernel abstraction, enabling execution on both multicore CPUs and GPU accelerators from a single code base. Performance analyses conducted on NVIDIA CUDA GPUs demonstrate substantial reductions in runtime relative to CPU execution, particularly for large-scale datasets involving up to approximately 3.3 million rectangular prisms, highlighting the scalability of the proposed approach. The inversion incorporates implicit model constraints through the data-space formulation and depth-weighted sensitivity, which mitigate depth-related amplitude decay and yield geologically coherent, high-resolution subsurface density models. Validation using synthetic models confirms the ability of the framework to accurately reconstruct complex subsurface structures such as vertical and dipping dykes. Application to field gravity data further demonstrates the robustness and practical utility of the GPU-accelerated framework, with the recovered models showing strong consistency with independent geological constraints and prior interpretations. Overall, this work underscores the potential of GPU-enabled computing in Julia to transform large-scale gravity inversion workflows, providing an efficient, extensible, and accurate computational solution for high-resolution geophysical studies.

## 摘要（中文）
arXiv:2602.03857v1 公告类型：新 摘要：本文提出了一种在 Julia 中实现的三维重力建模和反演的高性能框架，解决了地球物理建模中的关键挑战，例如计算复杂性、不适定性以及重力反演固有的非唯一性。该框架采用数据空间反演公式来降低问题的维度，从而在保持反演精度的同时显着降低内存需求并提高计算效率。正向建模和反演运算符在与后端无关的内核抽象中实现，从而可以从单个代码库在多核 CPU 和 GPU 加速器上执行。对 NVIDIA CUDA GPU 进行的性能分析表明，相对于 CPU 执行，运行时间显着减少，特别是对于涉及多达约 330 万个矩形棱柱的大规模数据集，凸显了所提出方法的可扩展性。反演通过数据空间公式和深度加权灵敏度纳入隐式模型约束，从而减轻与深度相关的振幅衰减并产生地质相干的高分辨率地下密度模型。使用合成模型的验证证实了该框架准确重建复杂地下结构（例如垂直堤坝和倾斜堤坝）的能力。现场重力数据的应用进一步证明了 GPU 加速框架的鲁棒性和实用性，恢复的模型与独立的地质约束和先前的解释表现出很强的一致性。总体而言，这项工作强调了 Julia 中支持 GPU 的计算在改变大规模重力反演工作流程方面的潜力，为高分辨率地球物理研究提供高效、可扩展且准确的计算解决方案。
