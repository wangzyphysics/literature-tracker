## Abstract
arXiv:2502.04298v2 Announce Type: replace Abstract: Continuous-time Markov chains have been successful in modelling systems across numerous fields, with currents being fundamental entities that describe the flows of energy, particles, individuals, chemical species, information, or other quantities. They apply to systems described by agents transitioning between vertices along the edges of a network (at some rate in each direction). It has recently been shown by the authors that, at stationarity, a hidden linearity exists between currents that flow along edges: if one controls the current of a specific "input" edge (by tuning transition rates along it), any other current is a linear-affine function of the input current [PRL 133, 047401 (2024)]. In this paper, we extend this result to the situation where one controls the currents of several edges, and prove that other currents are in linear-affine relation with the input ones. Two proofs with distinct insights are provided: the first relies on Kirchhoff's current law and reduces the input set inductively through graph analysis, while the second utilizes the resolvent approach via a Laplace transform in time. We obtain explicit expressions for the current-to-current susceptibilities, which allow one to map current dependencies through the network. We also verify from our expression that Kirchhoff's current law is recovered as a limiting case of our mutual linearity. Last, we uncover that susceptibilities can be obtained from fluctuations when the reference system is originally at equilibrium.

## 摘要（中文）
arXiv:2502.04298v2 公告类型：替换 摘要：连续时间马尔可夫链已成功地对多个领域的系统进行建模，其中电流是描述能量、粒子、个体、化学物质、信息或其他量的流动的基本实体。它们适用于由代理沿着网络边缘（在每个方向上以某种速率）在顶点之间转换所描述的系统。作者最近表明，在平稳状态下，沿边缘流动的电流之间存在隐藏的线性：如果控制特定“输入”边缘的电流（通过调整沿其的转换率），则任何其他电流都是输入电流的线性仿射函数 [PRL 133, 047401 (2024)]。在本文中，我们将这一结果推广到一个控制多个边的电流的情况，并证明其他电流与输入电流呈线性仿射关系。提供了两个具有不同见解的证明：第一个依赖于基尔霍夫现行定律并通过图形分析归纳减少输入集，而第二个则利用通过拉普拉斯时间变换的解析方法。我们获得了当前到当前敏感性的显式表达式，这允许人们通过网络映射当前的依赖关系。我们还从我们的表达式中验证了基尔霍夫电流定律被恢复为我们的相互线性的极限情况。最后，我们发现当参考系统最初处于平衡状态时，可以从波动中获得敏感性。
