## Abstract
arXiv:2602.11709v1 Announce Type: new Abstract: High-throughput powder X-ray diffraction (XRD) simulations are a key prerequisite for generating large datasets used in the development of machine-learning models for XRD-based materials analysis. However, the widely used pymatgen powder XRD calculator, implemented entirely in Python, can be computationally inefficient for large-scale workloads, limiting throughput. We present XRD-Rust, a Rust-accelerated implementation of the pymatgen powder XRD calculator that maintains full compatibility with existing Python-based workflows. The method retains pymatgen for crystal structure handling and symmetry analysis while reimplementing the computationally intensive parts of the XRD calculation in Rust. Performance benchmarks were carried out on large crystallographic datasets from the Materials Cloud Three-Dimensional Structure Database (MC3D, 33 142 structures) and the Crystallography Open Database (COD 515 181). For the MC3D dataset, XRD-Rust achieves an average speedup of 4.7 +- 1.6 and a maximum speedup of 25, reducing computation from 34.9 s to 1.4 s. For the COD dataset, the average speedup is 6.1 +- 4.6 with a maximum speedup of 719 (1437 min to 2 min). These benchmarks demonstrate that XRD-Rust significantly accelerates powder XRD simulations, enabling efficient high-throughput dataset generation and improved performance in interactive diffraction analysis applications.

## 摘要（中文）
arXiv:2602.11709v1 公告类型：新 摘要：高通量粉末 X 射线衍射 (XRD) 模拟是生成大型数据集的关键先决条件，该数据集用于开发基于 XRD 的材料分析的机器学习模型。然而，广泛使用的 pymatgen 粉末 XRD 计算器完全用 Python 实现，对于大规模工作负载来说计算效率可能较低，从而限制了吞吐量。我们推出 XRD-Rust，这是 pymatgen 粉末 XRD 计算器的 Rust 加速实现，它与现有的基于 Python 的工作流程保持完全兼容。该方法保留了 pymatgen 用于晶体结构处理和对称性分析，同时在 Rust 中重新实现 XRD 计算的计算密集型部分。对来自 Materials Cloud 三维结构数据库（MC3D，33 142 个结构）和晶体学开放数据库（COD 515 181）的大型晶体学数据集进行了性能基准测试。对于 MC3D 数据集，XRD-Rust 实现了 4.7 ± 1.6 的平均加速比和 25 的最大加速比，将计算时间从 34.9 秒减少到 1.4 秒。对于 COD 数据集，平均加速比为 6.1 ± 4.6，最大加速比为 719（1437 分钟到 2 分钟）。这些基准测试表明 XRD-Rust 显着加速了粉末 XRD 模拟，从而实现高效的高通量数据集生成并提高了交互式衍射分析应用的性能。
